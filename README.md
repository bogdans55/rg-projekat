# rg-projekat

Scene containing crossroad with four traffic lights and a car patiently waiting.
It was made in C++ using OpenGL, GLFW and GLAD. It contains:
- Shaders 
- Textures 
- Transformations 
- Coordinate systems 
- Camera 
- Colors 
- Basic lighting 
- Materials 
- Lighting maps 
- Light casters 
- Multiple lights 
- Assimp 
- Mesh 
- Models 
- Model and lighting 
- Imgui 
- Blending 
- Face culling 
- Advanced lighting 
- Cubemaps
- HDR and bloom

## Controls

`W`  - Move camera forward \
`S`  - Move camera backward \
`A`  - Move camera left \
`D`  - Move camera right \
`SPACE`  - Move camera up \
`LCONTROL`  - Move camera down \
`MOUSE`  - Look around \
`SCROLL`  - Zoom
